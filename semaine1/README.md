### **2. PREMIÈRES IDÉES (40%)**

**Format :** Brainstorming et questions

```markdown
# NOS PREMIÈRES RÉFLEXIONS

## L'idée de base :

"Nous allons créer une plateforme où les utilisateurs pourront :

- Uploader leurs vidéos
- Regarder des vidéos
- Créer un compte sécurisé"

## Nos questions techniques :

• "Comment stocker les vidéos ?"
• "Comment faire pour que les vidéos se chargent vite ?"
• "Comment protéger les comptes utilisateurs ?"
• "Où stocker les informations des vidéos ?"

## Ce qu'on imagine :

[Dessin à la main ou schéma simple montrant]
"Utilisateur → Site web → Stockage vidéos → Base de données"
```

## **CONSEILS CONCRETS**

### **Comment remplir ce rapport :**

```bash
1. "Prendre 30 minutes en équipe pour discuter"
2. "Chacun dit ce qu'il aimerait faire dans le projet"
3. "Noter toutes les questions qui viennent à l'esprit"
4. "Faire un simple schéma à la main"
5. "Envoyer le document au professeur"
```

### **Exemple de questions simples à poser :**

```markdown
- "Est-ce qu'on va programmer ou juste configurer ?"
- "Comment les utilisateurs vont-uploader leurs vidéos ?"
- "Où est-ce qu'on va stocker tout ça ?"
- "Comment on fait pour que ce soit sécurisé ?"
```
